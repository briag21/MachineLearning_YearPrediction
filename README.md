# Projet académique (Licence 3) de Machine Learning
Le rapport présente un projet réalisé dans le cadre du module Introduction à l'Intelligence Artificielle (Pr P. Hellier, Univ. Rennes). Ce projet est divisé en deux parties, régression et classification. Je me permets de partager ici uniquement la partie sur laquelle j'ai travaillé (en collaboration avec E. Abdoum et A. Touzet), la partie régression.

Le projet de régression consistait à trouver la meilleure méthode pour réaliser le défi Kaggle Year Prediction MSD.

Le compte rendu explique la méthode et les résultats obtenus dans le notebook, également présent dans le répertoire (regression_YearPrediction.ipynb).
